# Exercício – Login e Senha (C#)

Programa em C# Console Application que realiza validação de login e senha.

## Funcionalidades
- Login aceita apenas letras
- Senha aceita apenas números
- Validação segura usando TryParse e char.IsLetter

## Tecnologias
- C#
- .NET
- Console Application

## Objetivo
Praticar lógica de programação e boas práticas em C#.
